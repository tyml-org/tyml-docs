    enum TypeðŸ”—

    Declaring an enum can restrict string values:

    enum Mode {
    "Debug"
    "Release"
}

// "mode" only accepts "Debug" or "Release"
mode: Mode

    You can separate enum items with commas or newlines, just like struct fields:

    enum Mode {
    "Debug",
    "Mode",
}
